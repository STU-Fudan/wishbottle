# wishbottle

Still under developing...

## Build

Dev dependencies:

`npm`, `jade`, `browserify`, `uglifyjs`, `watch`, `live-reload`, `parallelshell` (check out package.json for more details).

- `$ npm install`
- `$ npm run build`
- open dist/index.html

## Develop

- build the project
- `$ npm run dev`
